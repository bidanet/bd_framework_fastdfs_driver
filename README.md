使用方法：

maven  pom中加入 

<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>


	<dependency>
	    <groupId>com.github.bidanet</groupId>
	    <artifactId>bd_framework_fastdfs_driver</artifactId>
	    <version>v0.1</version>
	</dependency>
